# BotFather Configuration

Use these settings to configure your bot via @BotFather.

## 1. Edit Commands
Send `/setcommands` to BotFather and paste the following list:

```text
start - Avvia il bot
menu - Pannello principale
status - Riepilogo e Statistiche
history - Storico e Cancellazione
log - Inserimento Manuale (Eventi passati)
import - Import Data da CSV
child - Profilo del Bambino
admin - Gestione Condivisione
help - Guida e Supporto
```
